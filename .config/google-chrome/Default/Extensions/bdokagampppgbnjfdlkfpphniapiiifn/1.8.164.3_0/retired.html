Options (formerly YouTube Options)
Important information for Chrome Web Store users.

Dear Options users,

You are seeing this message because you are using a version of Options that is no longer maintained or supported.  We have chosen to retire this old version of Options because we feel that leaving it broken and partially working in the Chrome Web Store is confusing and frustrating for existing users.  Following the April 2nd, 2014 announcement that Options was moving to a subscription model, new features have been added, a version for Mozilla Firefox has been released, and lots of changes have been made to deal with the constantly evolving YouTube pages.  We hope you choose to install the current release for Google Chrome or grab a different browser version from our homepage.

FAQs

Why the name change?

We needed to change the name due to changes in Chrome Web Store policy.  Most extensions with “YouTube” in their titles have experienced the same thing.

Why is there a different version to install?

Due to Chrome Web Store policy, we needed to create a separate version that you will need to install independently of this old Web Store version.

Where can I find the new version?

The current Chrome Web Store version can be found here.  Current versions for all supported browsers can always be found on the Options homepage.

Why are we charging for Options?

Since its release in 2010, Options has been a free product built to give people control over their online viewing experiences.  Starting on April 8, 2014, Options moved to a paid subscription model.  Licensed users are billed $1.99 monthly, with six-month and twelve-month plans also available.

Where do I get an Options license?

You can grab a subscription license now, or after installing the current release of Options you will find a link to the subscription license page.  You can also request a three-day trial license.

If I am having trouble with Options how do I get help?

For help with everything Options, please contact us using our website.  You will also find a “Help” button in the Options page-action menu, buttons in the “Help” tab in Options, and a link in the footer of most YouTube pages.
